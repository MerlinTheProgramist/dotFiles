# My config files 
managed with help of [chezmoi](https://www.chezmoi.io/),  
applying all of my configuration is as easy as:
```bash
chezmoi init --apply https://github.com/MerlinTheProgramist/dotFiles.git
```
---
contains configs for:
- bashrc (lots of helpful bash commands)
- [alacritty](https://alacritty.org/) (terminal emulator)
- [helix](https://helix-editor.com/) (postmodern vim like text editor)
- [ranger](https://ranger.fm/) (terminal file manager)
- [i3wm](https://i3wm.org/) (xorg tiling window manager)
- [sway](https://swaywm.org/) (wayland tiling window manager)
- [qutebrowser](https://qutebrowser.org/) (browser with vim keybindings)
- [git](https://git-scm.com/) (some helpful shortcuts)  

software that i use (special functions with the above suite):

- [zathura](https://pwmt.org/projects/zathura/) (document viewer)
- [rofi](https://github.com/davatorium/rofi) (dmenu, calc, emoji)  
I use arch btw.
